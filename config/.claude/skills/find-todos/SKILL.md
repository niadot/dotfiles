# Find TODOs スキル

コードベース内のTODO、FIXME、HACKなどのコメントを検索してレポートします。

## 説明
プロジェクト内の未完了タスクや改善点をすばやく特定します。

## 使い方
```
/find-todos [ディレクトリパス]
```

パスを指定しない場合は、カレントディレクトリを検索します。

## 指示

ユーザーがこのスキルを実行したとき、以下を実施してください：

1. **検索パターン**
   以下のパターンをコードから検索：
   - TODO
   - FIXME
   - HACK
   - XXX
   - NOTE
   - BUG

2. **検索の実行**
   - Grepツールを使用して、指定されたディレクトリ（またはカレントディレクトリ）を検索
   - テストファイル、ビルド成果物、node_modules、.gitなどは除外

3. **結果の整理**
   以下の形式で結果を表示：

   ```
   ## 📝 TODO/FIXME検索結果

   ### 🔴 優先度: 高 (FIXME, BUG)
   - `ファイル名:行番号` - コメント内容

   ### 🟡 優先度: 中 (TODO, HACK)
   - `ファイル名:行番号` - コメント内容

   ### 🔵 優先度: 低 (NOTE, その他)
   - `ファイル名:行番号` - コメント内容

   ### 📊 サマリー
   - 合計: X件
   - FIXME/BUG: Y件
   - TODO/HACK: Z件
   ```

4. **アクション提案**
   - 見つかったTODOの数が多い場合、優先順位付けを提案
   - すぐに対応可能な項目があれば指摘
